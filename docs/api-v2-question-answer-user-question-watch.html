获取关注的问题列表

GET /api/v2/user/question-watches

参数

参数类型描述
limitInteger默认 20 ，获取列表条数，修正值 1 - 30。
offsetinteger默认 0 ，数据偏移量，传递之前通过接口获取的总数。

关注一个问题

响应
